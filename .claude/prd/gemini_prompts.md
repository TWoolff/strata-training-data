# Gemini Character Generation Prompts

## Single-Character Prompts for Seg Training (SC-001+)

Simpler than turnarounds — one character per image, transparent/white background, full body visible. Used to expand segmentation training data via See-Through pseudo-labeling pipeline. Goal: style and pose diversity.

Each prompt is fully self-contained — paste it into Gemini as-is. Prompts are generated by `scripts/generate_character_prompts.py` which rotates through style × character × pose pools. Run the script to regenerate or extend.

```bash
python3 scripts/generate_character_prompts.py --count 100 --start_id 201 --output /tmp/more.md
```
