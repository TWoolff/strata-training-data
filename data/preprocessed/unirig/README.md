# UniRig / Rig-XL

14K rigged 3D meshes. SIGGRAPH 2025.

## Download

- **URL:** https://github.com/VAST-AI-Research/UniRig
- **Size:** ~20 GB
- **Method:** Clone repo, follow Rig-XL dataset download instructions

```bash
./ingest/download_datasets.sh unirig
```

Check repository license. Academic use.

## Format

Rigged 3D meshes with skeleton and skinning weights. Useful for bone mapping and weight prediction training.
