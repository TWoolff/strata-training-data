# AnimeRun

~8K contour/color frame pairs from anime videos. Used for contour line detection training.

## Download

- **URL:** https://github.com/lisiyao21/AnimeRun
- **Project page:** https://lisiyao21.github.io/projects/AnimeRun
- **Size:** ~5 GB
- **Method:** Clone repo, follow download links (Google Drive / Baidu)

```bash
./ingest/download_datasets.sh animerun
```

## Format

Paired anime frames with extracted contour lines. Frame sequences from anime productions.
