# Character FBX Files

Source characters (.fbx) for the segmentation pipeline.

## Sources

- **Mixamo**: Download from https://www.mixamo.com/ (free account required)
- **Sketchfab**: CC0/CC-BY characters from https://sketchfab.com/
- **Quaternius**: CC0 low-poly characters from https://quaternius.com/
- **Kenney**: CC0 game assets from https://kenney.nl/

## Expected Structure

Place .fbx files directly in this directory:

```
data/fbx/
├── mixamo_001.fbx
├── mixamo_002.fbx
├── quaternius_robot.fbx
└── ...
```
