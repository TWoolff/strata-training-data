# Pose Library

Animation FBX clips used to pose characters during rendering.

## Sources

- **Mixamo**: Download animation clips from https://www.mixamo.com/
- Select "Without Skin" to get animation-only FBX files

## Expected Structure

Place .fbx animation files directly in this directory:

```
data/poses/
├── Walking.fbx
├── Running.fbx
├── Idle.fbx
└── ...
```
